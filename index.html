HTML

HyperText Markup Language adalah suatu bahasa menggunakan tag tertentu yang ditafsirkan browser.

CSS

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur.

JS

JavaScript adalah bahasa pemrograman tingkat tinggi dan dinamis. Kode JavaScript dapat disisipkan dalam halaman web menggunakan tag SCRIPT.

PHP

Hypertext Preprocessor adalah bahasa skrip dengan fungsi umum yang terutama digunakan untuk pengembangan web.
